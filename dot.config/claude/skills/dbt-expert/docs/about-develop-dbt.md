On this page

Develop dbt projects using the dbt platform, a faster and more reliable way to deploy dbt and manage your project in a single, web-based UI.

You can develop in your browser using a dbt-powered command line interface (CLI), an integrated development environment (Studio IDE), or Canvas.

## Getting started[â€‹](#getting-started "Direct link to Getting started")

To get started, you'll need a [dbt](https://www.getdbt.com/signup) account and a developer license. For a more comprehensive guide about developing in dbt, refer to the [quickstart guides](https://docs.getdbt.com/docs/get-started-dbt).

Choose the option that best fits your needs:

[![](https://docs.getdbt.com/img/icons/dbt-bit.svg)

#### dbt CLI

Allows you to develop and run dbt commands from your local command line or code editor against your dbt development environment.](/docs/cloud/cloud-cli-installation)

[![](https://docs.getdbt.com/img/icons/vsce.svg)

#### dbt VS Code extension

Bring the speed and intelligence of the dbt Fusion engine to VS Code for a seamless local development experience.](/docs/about-dbt-extension)

[![](https://docs.getdbt.com/img/icons/dashboard.svg)

#### dbt Studio IDE

Develop dbt projects directly in your browser with seamless SQL compilation and an intuitive, visual workflow.](/docs/cloud/studio-ide/develop-in-studio)

[![](https://docs.getdbt.com/img/icons/canvas.svg)

#### dbt Canvas

Develop with Canvas, a seamless drag-and-drop experience that helps analysts quickly create and visualize dbt models in dbt.](/docs/cloud/canvas)
