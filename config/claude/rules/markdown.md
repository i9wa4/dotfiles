# Markdown Rules

## 1. Universal Rules

- NEVER: Do not use emojis
- NEVER: Do not start numbered lists from 0 (tools auto-convert 0 to 1)
- YOU MUST: Align table columns with spaces

Good example:

```markdown
| Name   | Description | Value |
| ------ | ----------- | ----- |
| foo    | Foo item    | 100   |
| barbaz | Bar baz     | 200   |
```

Bad example:

```markdown
| Name | Description | Value |
| --- | --- | --- |
| foo | Foo item | 100 |
| barbaz | Bar baz | 200 |
```

## 2. Japanese Markdown Rules

- NEVER: Do not use bold
- NEVER: Do not use trailing colons (:)
